# 🍽️ SmartCanteen

SmartCanteen adalah aplikasi pemesanan makanan kantin berbasis mobile yang dikembangkan menggunakan Flutter dan Firebase. Aplikasi ini membantu pengguna memesan makanan secara online, melakukan pembayaran digital, melacak pesanan secara realtime, serta mendapatkan rekomendasi makanan berbasis AI.

## ✨ Fitur

### User
- Login & Register
- Reset Password
- Melihat Menu Makanan
- Pencarian Menu
- Keranjang Belanja
- Checkout Pesanan
- QR Payment
- Tracking Pesanan Realtime
- Chat Customer Service
- Notifikasi
- Profil Pengguna
- Rekomendasi Makanan AI

### Admin
- Dashboard Admin
- Kelola Menu
- Kelola Kategori
- Kelola Pesanan
- Chat User
- Statistik Penjualan
- Notifikasi Admin
- Pengaturan Kantin

## 🛠️ Teknologi

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Cloud Messaging
- Firebase Storage
- Artificial Intelligence Recommendation

## 🚀 Instalasi

Clone repository:

```bash
git clone https://github.com/username/SmartCanteen.git
```

Masuk ke folder project:

```bash
cd SmartCanteen
```

Install dependency:

```bash
flutter pub get
```

Jalankan aplikasi:

```bash
flutter run
```
